This quickstart walks through a minimal Tandemn setup: an administrator starts the server, a user connects the CLI, and the user submits a batch inference job.

<Note>
  You need a network-reachable machine for the Tandemn server and Docker Compose support on that machine.
</Note>

## What you will do

<Steps>
  <Step title="Download the Tandemn server release">
    Pull the release bundle that contains the Docker Compose configuration and example environment file.
  </Step>
  <Step title="Start the server">
    Configure `.env`, pull the container images, and start the service stack.
  </Step>
  <Step title="Install the CLI">
    Install the `tandemn` Python package in your local environment.
  </Step>
  <Step title="Submit a job">
    Point the CLI at the server and run a first inference job.
  </Step>
</Steps>

## 1. Start the server

The Tandemn server is packaged as a release bundle with a `docker-compose.yml` file and an `.env.example` file.

```bash
curl -L -O "https://github.com/Tandemn-Labs/tandemn-system/releases/download/0.0.1/tandemn-0.0.1.tar.gz"
tar -xzf tandemn-0.0.1.tar.gz
cd tandemn-0.0.1
```

Copy the example environment file and edit the values for your cluster.

```bash
cp .env.example .env
```

Start the containers.

```bash
docker compose pull
docker compose up
```

<Tip>
  Keep the server URL handy. CLI users will set it as `TD_SERVER_URL`.
</Tip>

## 2. Install the CLI

Install the Tandemn CLI in a Python environment.

<CodeGroup>

```bash pip
pip install tandemn
```

```bash venv
python -m venv .venv
source .venv/bin/activate
pip install tandemn
```

</CodeGroup>

## 3. Connect to the server

Set the server URL and verify that the CLI can reach it.

```bash
export TD_SERVER_URL=<your-server-url>
tandemn check
```

If the check fails, see [Troubleshooting](/admin/troubleshooting).

## 4. Submit a job

Create or choose a JSONL prompt file, then submit it with a model and SLO.

```bash
tandemn deploy Qwen/Qwen2.5-7B-Instruct <prompts.jsonl> --slo 4
```

You should see CLI output showing that the job was accepted by the server.
