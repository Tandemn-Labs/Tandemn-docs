If you are blocked, include enough context for someone to reproduce the issue.

## For CLI users

Send your administrator:

- The command you ran
- The error message
- Your `TD_SERVER_URL` value, if it is not sensitive
- The model name
- A small non-sensitive sample of the prompt file

## For administrators

Check:

- The server containers are running
- The `.env` file is configured for the current deployment
- Users can reach the server URL over the network
- The requested model is supported
- The cluster has compatible capacity

## Contact Tandemn

For product or account help, email [contact@tandemn.com](mailto:contact@tandemn.com).
